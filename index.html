Demo Website | Git Repository

Smart Coffee Mug

  • I love coffee, so I decided to make a machine to help drink my coffee safely. The Smart Coffee Mug was developed using an Arduino board programmed in C++ using Onewire libraries to control its components.
  • It consists of a Coffee Cup, a Servo Motor, Temperature Sensor, Touch Sensor and Speaker.
  • First and foremost the mug constantly samples and displays the temperature of the beverage in the mug. It opens the lid of the cup whenever a user picks up the contraption. Furthermore, it keeps the lid open for the beverage inside the mug to cool if the beverage is too hot. Finally, it alerts users who pick up the contraption when it is too hot.
